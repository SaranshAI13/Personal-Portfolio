# Personal Portfolio 🌐

A premium, interactive personal portfolio website showcasing quantitative finance projects, software engineering implementations, and machine learning models. 

**Live Demo:** [saranshai13.github.io/Personal-Portfolio](https://saranshai13.github.io/Personal-Portfolio/)

## 🚀 Key Features

* **Interactive Particle System:** Custom constellation rendering canvas that repels nodes dynamically based on real-time cursor coordinates.
* **Custom Crosshair Cursor:** Double-ring fluid cursor tracking mouse movements with smooth easing and hover expansions.
* **Boot-Up Loader:** System initialization terminal simulation displaying loading percentage progress bar.
* **First-Person Bio:** Authentic summary of projects, education at DTU (Mathematics & Computing), and positions of responsibility.
* **Fully Responsive:** Completely optimized for mobile viewports, tablets, and desktop displays.

## 🛠️ Built With

* **Markup & Core Structure:** HTML5 (Semantic elements)
* **Styling & UI Effects:** Vanilla CSS3 (Custom properties, grid layouts, flexbox, keyframe animations)
* **Logic & Animations:** Modern JavaScript (ES6+, HTML5 Canvas API, Intersection Observer API)
* **Vector Graphics:** FontAwesome & DevIcons
